# SkyproWallet

Веб-приложение для учета личных расходов с возможностью регистрации, авторизации и анализа трат.

## Описание

SkyproWallet - это современное веб-приложение для управления личными финансами. Пользователи могут регистрироваться, входить в систему, добавлять расходы по категориям и анализировать свои траты.

## Установка и запуск

### Требования
- Node.js (версия 16 или выше)
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Создать .env/.env.development
*см. .env.example
```bash
    VITE_URL=address:api
```

### Запуск в режиме разработки
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Сборка для продакшена
```bash
npm run build
```

## Ключевые функции

- **Авторизация и регистрация** - безопасный вход в систему с валидацией данных
- **Управление расходами** - добавление, просмотр и удаление транзакций
- **Категоризация** - распределение расходов по категориям
- **Аналитика** - визуализация трат и статистика
- **Адаптивный дизайн** - корректная работа на всех устройствах

## Технологии

- **Frontend**: React 18, Vite
- **Роутинг**: React Router
- **Стилизация**: CSS Modules, CSS
- **Управление состоянием**: React Context API, useReducer
- **HTTP-клиент**: Fetch API
- **Хранение данных**: LocalStorage
- **Сборка**: Vite
- **Линтинг**: ESLint

## API

Приложение использует API Skypro для аутентификации и управления пользователями:
- Базовый URL: `https://wedev-api.sky.pro/api`
- Эндпоинты: `/user/login`, `/user`, `/user/me`
